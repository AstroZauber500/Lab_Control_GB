\section{Objetivo}

\begin{frame}{Objetivo}
\begin{itemize}
\item Identificar experimentalmente la planta del sistema de grúa (carro-péndulo).
\item Obtener un modelo SIMO promedio en espacio de estados.
\item Diseñar y evaluar controladores por realimentación de estados con acción integral (ISF):
\begin{itemize}
\item Pole Placement (ubicación de polos)
\item LQR (Regulador Cuadrático Lineal)
\end{itemize}
\item Comparar desempeño en simulación y experimento.
\item Evaluar atenuación de oscilación del péndulo.
\end{itemize}
\end{frame}

%------------------------------------------------
\section{Resumen}

\begin{frame}{Resumen}
\begin{itemize}
\item Identificación experimental de funciones de transferencia para posición del carro y ángulo del péndulo.
\item Procesamiento de datos mediante MATLAB System Identification Toolbox.
\item Construcción de un modelo SIMO promedio en espacio de estados.
\item Diseño de controladores ISF mediante Pole Placement y LQR.
\item Evaluación comparativa entre simulación y resultados experimentales.
\end{itemize}
\end{frame}

\begin{frame}{Obtención de datos}
\begin{itemize}
\item Plataforma: carro accionado por motor DC con transmisión polea-banda.
\item Péndulo acoplado al carro.
\item Medición mediante encoders ópticos incrementales:
\begin{itemize}
\item Posición del carro $p$
\item Ángulo del péndulo $\theta$

\item Excitación: entrada de voltaje trapezoidal de aproximadamente 7.5 V.
\item Periodo de muestreo: 20 ms.
\end{itemize}

\section{Controladores}

\begin{frame}{Controladores utilizados}
\begin{itemize}
\item Estrategia: Realimentación de estados con acción integral (ISF).
\item Se añade un estado integral asociado al error de seguimiento de posición:
\[
\dot{x}_I = r - C_r x
\]
\item El sistema aumentado tiene matriz de controlabilidad de rango 5 (totalmente controlable).
\item Dos técnicas evaluadas:
\begin{itemize}
\item Pole Placement (ubicación de polos)
\item LQR (Regulador Cuadrático Lineal)
\end{itemize}
\end{itemize}
\end{frame}

%------------------------------------------------
\subsection{Pole Placement}

\begin{frame}{Pole Placement}
\begin{itemize}
\item Diseño basado en ubicación deseada de polos en lazo cerrado.
\item Sistema aumentado con estado integral para anular error estacionario.
\item Cálculo de la ganancia $K$ mediante asignación directa de polos.
\item Permite garantizar dinámica dominante y atenuación del péndulo según los polos seleccionados.
\end{itemize}
\end{frame}

\section{Conclusiones}

\begin{frame}{Conclusiones}
\begin{itemize}
\item Se identificó la planta del sistema grúa obteniendo un modelo SIMO en espacio de estados representativo.
\item El controlador LQR cumplió las especificaciones tanto en simulación ($t_s = 4.70$ s, $M_p = 0.60\%$) como en experimento ($t_s = 5.25$ s, $M_p = 0\%$).
\item El error estacionario nulo se logró gracias a la acción integral del esquema ISF.
\item Los resultados experimentales son consistentes con la simulación; el ligero aumento del $t_s$ se debe a fricción, zona muerta del motor y saturación del actuador.
\item La ausencia de sobreimpulso experimental sugiere mayor amortiguamiento físico que el del modelo lineal.
\end{itemize}
\end{frame}

%------------------------------------------------
\subsection{Recomendaciones}

\begin{frame}{Recomendaciones}
\begin{itemize}
\item Incorporar más sets de datos experimentales para reducir el CV en la identificación.
\item Descartar muestras atípicas con criterios estadísticos formales.
\item Considerar efectos no lineales (fricción, zona muerta, saturación) en el modelo.
\item Ajustar los pesos de Bryson de forma conservadora para reducir esfuerzo de control.
\item Explorar técnicas robustas para mejorar la atenuación de oscilación del péndulo.
\end{itemize}
\end{frame}
